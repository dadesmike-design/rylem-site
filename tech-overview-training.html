Module 1

The Basics: Hardware vs Software

Every tech role falls into one of these two worlds — or the bridge between them.

🖥️

Hardware

The physical stuff you can touch. Servers, laptops, network switches, CPUs, GPUs. When a company says "infrastructure," they usually mean hardware + the software that runs it.

Recruiter Tip

Hardware roles (Network Engineer, Systems Admin, Data Center Tech) are often on-site. Remote is rare. Pay attention to location requirements.

💻

Software

The programs, apps, and systems that run on hardware. Everything from your phone's apps to Netflix's recommendation engine to a company's internal tools.

Recruiter Tip

Software roles are the bulk of tech hiring. They're more likely to be remote-friendly. Focus on languages, frameworks, and cloud experience.

Module 4

Tech Roles Explained

The same person might be called different titles at different companies. Focus on what they DO, not what they're called.

🏗️ Software Architect

Designs the overall system. Picks technologies. Makes decisions that affect the whole engineering org. Rarely writes daily code anymore.

$180K–$280K+ | Senior/Staff level

👨‍💻 Software Engineer / Developer

Writes the actual code. Builds features, fixes bugs, writes tests. The backbone of any engineering team. Levels: Junior → Mid → Senior → Staff → Principal.

$90K–$250K+ | All levels

🎨 Frontend Engineer

Specializes in what users see — buttons, pages, animations. Works with React, CSS, TypeScript. Partners closely with designers.

$100K–$200K | Mid-Senior

⚙️ Backend Engineer

Builds the server-side logic, APIs, and database interactions. The "behind the scenes" work. Python, Java, Go, Node.js.

$110K–$220K | Mid-Senior

🔧 Full-Stack Engineer

Can work on both frontend AND backend. Jack of all trades. Very common at startups where teams are small.

$110K–$230K | Mid-Senior

☁️ DevOps / SRE / Platform Engineer

Keeps systems running. Manages cloud infrastructure, deployments, monitoring. Site Reliability Engineer (SRE) is Google's version. Hottest hiring category in 2026.

$130K–$250K | Mid-Senior

📊 Data Engineer

Builds pipelines that move and transform data. Works with Spark, Airflow, Snowflake, dbt. Different from Data Scientist (who analyzes data).

$120K–$220K | Mid-Senior

🤖 AI / ML Engineer

Builds machine learning models and AI features. Python, PyTorch, LLMs, prompt engineering. The fastest-growing role category in tech right now.

$150K–$350K+ | Mid-Senior

🔒 Security Engineer

Protects systems from hackers. Penetration testing, compliance, incident response. Every company needs them — most are understaffed.

$130K–$250K | Mid-Senior

🛠️ Systems / IT Administrator

Manages internal tools, networks, servers. Keeps the lights on. Often works with Active Directory, Office 365, networking gear.

$70K–$140K | Junior-Senior

📋 Product Manager (Technical)

Not an engineer, but works with engineers daily. Decides WHAT to build and WHY. Bridges business and technology. Often needs technical literacy.

$120K–$220K | Mid-Senior

🧪 QA / Test Engineer

Makes sure software works before it ships. Manual testing → automated testing → SDET (writes code to test code). Critical but often overlooked.

$80K–$170K | All levels

Recruiter Tip — Agile Vocabulary

Sprint: A 1-4 week work cycle (usually 2 weeks)
Standup: Daily 15-min team check-in
Scrum Master: Facilitates the Agile process (not a manager)
Product Owner: Decides what to build next
Backlog: The prioritized list of things to build
Kanban: Alternative to Scrum — continuous flow, no sprints

Module 6

Cloud, AI & DevOps

The three biggest shifts in tech hiring. If your candidate knows these, they're in demand.

☁️

Cloud Computing

Instead of buying servers, companies rent computing power from AWS, Azure, or Google Cloud. Almost every modern company has moved to cloud.

AWS (Amazon) — #1 market share Azure (Microsoft) — #2, big in enterprise GCP (Google) — #3, strong in data/AI
Recruiter Tip

AWS certifications (Solutions Architect, Developer) are the most in-demand cloud certs. Azure certs are key for Microsoft shops. Ask candidates which cloud they've worked in — it matters.

🔄

DevOps & CI/CD

DevOps = Development + Operations working together. CI/CD = Continuous Integration / Continuous Deployment — automating the process of shipping code. Companies that do this well deploy new features multiple times per day.

Docker — packages apps in containers Kubernetes (K8s) — orchestrates containers Terraform — infrastructure as code Jenkins / GitHub Actions — automation
🤖

AI & Machine Learning

The hottest area in tech. Every company is adding AI features. Large Language Models (LLMs like ChatGPT) have created entirely new role categories.

Python — the #1 AI language PyTorch / TensorFlow — ML frameworks LLMs — Large Language Models RAG — Retrieval Augmented Generation Vector DBs — Pinecone, Weaviate MLOps — deploying ML to production
Recruiter Tip

AI/ML roles pay 20-40% more than equivalent software roles. If a candidate has production ML experience (not just Kaggle), they're extremely valuable.

Module 7

Recruiter Cheat Sheet

Quick reference cards. Print these. Pin them. Use them on every intake call.

🗣️ Intake Call Questions (Tech Roles)

  • "What's your tech stack?" (frontend, backend, cloud)
  • "Is this a new role or backfill?"
  • "Agile or Waterfall? Scrum or Kanban?"
  • "What cloud platform?" (AWS / Azure / GCP)
  • "Monolith or microservices?"
  • "What's the team size and who do they report to?"
  • "Any required certifications?"
  • "CI/CD pipeline — what tools?"

🔍 Resume Red/Green Flags

  • ✅ Green: GitHub profile with real contributions
  • ✅ Green: Specific tech versions (React 18, .NET 8)
  • ✅ Green: Quantified impact ("reduced latency 40%")
  • ✅ Green: Cloud certifications (AWS SA, Azure)
  • 🚩 Red: Lists 20+ technologies with no depth
  • 🚩 Red: "Proficient in all programming languages"
  • 🚩 Red: No measurable achievements
  • 🚩 Red: Only lists outdated tech (jQuery, COBOL)

💰 2026 Salary Ranges (US, W2)

  • Junior SWE: $85K–$130K
  • Mid SWE: $130K–$180K
  • Senior SWE: $170K–$250K
  • Staff/Principal: $220K–$350K+
  • DevOps/SRE: $140K–$260K

🔤 Acronym Decoder

  • API: Application Programming Interface (how systems talk)
  • CI/CD: Continuous Integration / Continuous Deployment
  • SaaS: Software as a Service (cloud apps)
  • SDLC: Software Development Lifecycle
  • SRE: Site Reliability Engineering
  • LLM: Large Language Model (ChatGPT, etc.)
  • K8s: Kubernetes (container orchestration)
  • IaC: Infrastructure as Code (Terraform)
  • RAG: Retrieval Augmented Generation (AI + search)
  • SDET: Software Dev Engineer in Test

🏢 Common Stack by Company Type

  • Microsoft ecosystem: C#, .NET, Azure, SQL Server, Teams
  • Startup (Series A-C): React, Node/Python, AWS, PostgreSQL
  • Enterprise (F500): Java, Spring, Oracle, on-prem + cloud hybrid
  • AI company: Python, PyTorch, AWS/GCP, Snowflake, vector DBs
  • Fintech: Java/Go, Kafka, PostgreSQL, strict security
  • Gaming: C++, Unreal/Unity, cloud backend, GPU

📞 Candidate Screening Questions

  • "Walk me through a project you built end-to-end"
  • "Which cloud have you deployed to in production?"
  • "Do you prefer frontend, backend, or full-stack?"
  • "What's your experience with CI/CD pipelines?"
  • "Have you worked in Agile? What was your sprint cadence?"
  • "What's the largest team / codebase you've worked in?"
  • "Any experience with containerization (Docker/K8s)?"
  • "What's your ideal next role?"
